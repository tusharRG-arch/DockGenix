Accessibility

A website designed for different ways of navigating

DockGenix aims to make its public website usable with keyboards, assistive technology, zoom, reduced motion, and smaller screens.

Current support

Accessibility is part of the component system

  • Skip link and semantic page landmarks
  • Logical heading structure
  • Keyboard-accessible navigation
  • Visible focus indicators
  • Reduced-motion support
  • Responsive text and layout
  • Text alternatives for meaningful images
  • Native controls for tables and disclosures

Target

WCAG 2.2 Level AA

The redesign targets WCAG 2.2 AA, but automated and manual testing must continue as new scientific figures, forms, and content are added.

If a page or interaction creates a barrier, email info@dockgenix.in with the page address, the problem encountered, and the assistive technology or browser used if you are comfortable sharing it.
